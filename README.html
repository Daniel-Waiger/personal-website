Personal Portfolio Website

Welcome to my personal portfolio website! This site showcases my work and skills in Microscopy, Image Analysis, and 3D Design.

Features

  • Responsive Design: Optimized for various devices and screen sizes.
  • Hero Section: Featuring a tagline, image, and a call-to-action button.
  • Navigation: Easy access to About, Portfolio, Services, and Contact pages.
  • Social Media Links: Connect with me on GitHub and LinkedIn .
  • Modern Styling: Utilizes custom CSS variables and Font Awesome icons.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • React

Installation

  1. Clone the repository: 
    git clone https://github.com/Daniel-Waiger/personal-website.git
  2. Navigate to the project directory: 
    cd personal-website
  3. Open index.html in your browser:

    You can open the file directly or use a local server for better performance.
